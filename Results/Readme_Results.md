## Readme
This folder contains the outputs of: 
- Differential Gene Expression Analysis (DGE_Results_SCZ.zip)
- Over-representation Analisys (ORA_Results_SCZ.zip)
- Gene Set Enrichment Analisys (GSEA_Results_SCZ.zip)

In addition, XLSX files are the filtered results of DGE Analyses according to the criteria of [Loo et al., 2024] (https://academic.oup.com/bib/article/25/6/bbae499/7820066):
- Sepecific_genes. xlsx: genes with a p_value > 0.5 in one sex and an adj_p_value < 0.05 in the other sex
- Dimorphic_genes.xlsx: genes with an adj_p_value < 0.05 in both sexes and an opposite logfoldchange > |0.5|
- Cell_type_Abundance.xlsx: Tabular data detailing cell-type proportions
