## Readme_Python
Complete workflow of snRNA-seq:
1. Input of Datasets
2. Preprocessing data: quality control, normalization, scale data
3. Dimensionality reduction - Principal Component Analysis (PCA) - and Integration - Harmony and scVI -.
4. Clustering and Benchmarking
5. Cell type annotation, Differential Gene Expression Analysis and Differential Abundance Analysis
6. Funtional Enrichment Analysis
